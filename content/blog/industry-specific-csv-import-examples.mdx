As the founder of Shelf, I've seen firsthand how proper asset management can transform operations across various industries. One of the most powerful features of Shelf is its ability to import existing asset data via CSV files. To help you get started, we've created a set of industry-specific CSV templates that you can use as a starting point for your own asset management journey.

## Why Use CSV Templates?

- **Quick Start:** These templates provide a structure that aligns with Shelf's import requirements, saving you time and reducing errors.
- **Industry-Specific Fields:** Each template includes fields relevant to its respective industry, giving you ideas for what to track.
- **Customization:** Use these as a base and add your own custom fields to suit your unique needs.

### How to Use These Templates

- Download the CSV file most relevant to your industry.
- Open the file in a spreadsheet program like Microsoft Excel or Google Sheets.
- Replace the sample data with your own asset information.
- Save the file as a CSV.
- Import the file into Shelf using our [CSV import feature](/knowledge-base/how-to-prepare-a-spreadsheet-for-bulk-importing-assets).

### Customizing Your CSV

While these templates provide a solid starting point, you may need to add or modify fields to match your specific needs. Shelf's flexible [custom fields](/knowledge-base-category/custom-fields) feature allows you to track any type of information relevant to your assets.

### Best Practices for CSV Imports

- Ensure your data is clean and consistent before importing.
- Use unique identifiers for each asset (e.g., serial numbers or asset tags).
- Standardize data formats, especially for dates and currency values.
- Consider using our [QR code asset tags](/asset-tags) for easy scanning and updates.
