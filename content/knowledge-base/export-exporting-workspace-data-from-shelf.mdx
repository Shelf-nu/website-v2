Shelf provides multiple ways to export your asset data, giving you flexibility in managing and analyzing your inventory information. Whether you need to export your entire workspace or just a specific selection of assets, our export features help you get the data you need in CSV format.

## Export Methods

### 1. Export Selected Assets

- Navigate to the Assets index
- Select the assets you want to export using the checkboxes
- Click the **Export** button at the top of the asset list
- Your selected assets will be downloaded as a CSV file

### 2. Asset Backup (Workspace Export)

- Go to Workspace settings
- Under the **General** tab, locate the **Download CSV** button
- Click to download a complete backup of all workspace assets


![2. Asset Backup (Workspace Export)](https://cdn.prod.website-files.com/641c35b7e5057648c76fa79f/670402b1e7c7eb985311550e_6703fcaab8050ddba68c9c10_Screenshot%25202024-10-07%2520at%252017.20.56.jpeg)
**Important Note:** QR codes are not included in the backup export. Due to the nature of how Shelf's QR codes work, they currently cannot be exported with assets because they have unique IDs. If you restore a backup, new QR codes will be generated for each asset. For backup restoration assistance, please contact our support team.

### 3. Export Filtered Results

- Apply any filters you need in the Assets index (e.g., by category, location, tags)
- Use the **Select all** option to select all filtered results
- Click **Export** to download the filtered selection

## What is Included in the Export

The CSV export includes comprehensive asset information:

- Asset names
- Descriptions
- Categories
- Tags
- Locations
- Valuations
- Custom fields
- Status information
- Custody details
- Activity history

## Using Exported Data

The exported CSV file can be used for:

- Data backup and archiving
- Analysis in spreadsheet applications
- Importing into other systems
- Creating reports
- Audit purposes
- **Bulk updates** — edit records in Excel or Google Sheets and re-import to apply changes at scale

### Bulk Updating via Export + Re-Import

A common workflow is to export a filtered selection of assets from the **Asset Index**, edit the CSV in a spreadsheet application, and re-upload it to update all the records at once. The **Update Existing Assets** feature handles this with a preview step that shows every field change before anything is saved.

> **Important:** Only use **Asset Index exports** for re-import updates — not the Workspace Backup CSV from Settings. Backup exports include internal fields not intended for re-import, and uploading one as an update file can corrupt data. The [CSV import guide](/knowledge-base/importing-assets-to-shelf-csv-guide) also notes this restriction.
