Finding the right items on a large booking just got easier. The asset list on a booking now shows a **Location** column, so you can see where each item is stored without leaving the booking — and you can click the column header to sort the list by location, which is handy for grouping a pull list by room or shelf before checkout.

The search bar above the list now reaches further too. As well as name, asset ID, and category, it matches an asset's **location**, **tags**, and its **QR and barcode values**. Separate terms with a comma to match any of them, and if a match lands inside a kit, the whole kit stays in view so the bundle isn't broken up.

Finally, the **Generate PDF** overview now respects whatever you've searched for. Narrow the list down first, generate the PDF, and you get a focused pull list of just the matching items instead of the entire booking — useful when you only need to hand off the gear from one location or one category.
