Shelf now supports four additional barcode formats — **Code128**, **Code39**, **Data Matrix**, and **EAN-13** — alongside the standard QR codes automatically generated for every asset. Organizations can integrate their existing barcode infrastructure into Shelf without re-tagging equipment.

Barcodes can be added to assets by typing the code manually or scanning an existing physical label with a device camera. Multiple barcodes can be attached to a single asset with no limit, and all formats work identically within Shelf's scanning, booking, and check-in systems.
