You can now update existing asset records in bulk using a CSV file. The workflow mirrors how most teams already manage data: export your assets, edit them in Excel or Google Sheets, and upload the file back to Shelf.

Before any changes are applied, Shelf shows you a **spreadsheet-style preview** with every field that will change — old value on the left, new value on the right. You confirm with "I AGREE" and the changes apply. Up to 1,000 assets per batch are supported.

You can update asset names, categories, locations, tags, valuations, availability, and any custom fields. Empty cells clear existing values, so you have full control over what gets removed. Categories, locations, and tags that do not yet exist are created automatically on apply.

This is the same plan tier as regular CSV import. Admins and Owners can access the workflow at **Assets → Import → Update existing**.
